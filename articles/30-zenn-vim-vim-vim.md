# まず、Vimをインストールしよう

## Mac
デフォルトでインストールされてます(だって、**標準のエディター**だからね)
ターミナルで、
```zsh
vim
```
って入力して何も出てこない場合、Homebrewでインストールしましょう。
```zsh
brew install vim
```

## Windows
~~導入結構めんどくさい~~Windows 11だと、デフォルトでwinget入っているらしい
```powershell
winget install Vim.Vim
```

知らぬ間に便利になってる
